# mnist-cnn-digit-classifier
CNN-based handwritten digit classification using TensorFlow/Keras on the MNIST dataset with ~99% accuracy
# Features
- Loads MNIST handwritten digit dataset (60,000 training images)
- Preprocesses image data (normalization and reshaping)
- Builds a Convolutional Neural Network (CNN) using TensorFlow/Keras
- Trains the model for digit classification (0–9)
- Evaluates model performance on test data
- Achieves ~99% validation accuracy
- Saves trained model for future predictions (.keras format)
